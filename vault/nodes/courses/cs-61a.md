CS 61A introduces the fundamental ideas of computer science using Python, Scheme, and SQL. Rather than dwelling on syntax, the course emphasizes **abstraction** — building complex programs by composing simple, well-understood pieces. Students learn to reason about programs as data and data as programs.

Core topics include higher-order functions, [[recursion]] and recursive data structures like the [[tree]], mutable state and object-oriented programming, and interpreters. The Scheme project, in which students write an interpreter for a subset of Scheme, ties these ideas together.

As the gateway course for the EECS major, CS 61A is the prerequisite for [[cs-61b]] and shapes how students think about every course that follows. It pairs computational fluency with the conceptual depth of the classic SICP curriculum.
