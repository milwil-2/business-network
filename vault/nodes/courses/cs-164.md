CS 164 covers the theory and engineering of programming languages by having students build a working compiler end to end. The pipeline spans lexical analysis, parsing, semantic analysis, and code generation.

Topics include regular expressions and finite automata for tokenizing, context-free grammars and parsing algorithms, type checking and scoping, and intermediate representations. The interpreter and language-design ideas first seen in [[cs-61a]] are revisited with full rigor.

Resting on the data structures of [[cs-61b]] and the machine model of [[cs-61c]], CS 164 gives students a deep understanding of how high-level code becomes something a processor can run.
