CS 186 looks inside the database to understand how it stores, indexes, and queries data efficiently. Students implement core components of a database engine, gaining hands-on experience with the algorithms that production systems use.

Topics include disk-based storage and buffer management, B+ tree [[index]] structures, join and sort algorithms, query optimization, and the ACID [[transaction]] guarantees enforced by concurrency control and recovery. The course uses [[postgres]] to ground these ideas, and contrasts the relational model with the [[property-graph-model]].

Building on the data structures of [[cs-61b]], CS 186 is the database systems course and a natural lead-in to data-intensive and systems work.
